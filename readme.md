Project Overview

This project is a basic front-end website called LearnFlow, made as part of my web development assignment. The website represents a simple online learning platform where users can view courses, check pricing plans, and contact the platform through a form.

The main goal of the project was to create a clean and responsive webpage using HTML and CSS, following the instructions given in the assignment.

Features

Simple and clean homepage

Header with navigation links

Hero section with an email signup form

Courses section showing three basic course cards

Pricing section with Free and Pro plans

Contact form for messages

Footer with basic links

Fully responsive layout for mobile screens

Files Included

index.html – Structure and content of the website

style.css – Styling and layout

images folder – Contains the hero image and any additional images

README.md – Documentation of the project

How to Run

Download the project folder.

Make sure the index.html, style.css, and images folder are in the same directory.

Open the index.html file in any browser (Chrome, Edge, Firefox, etc.).

The website will load and display all sections normally.
